      Creates a new identifier based on the provided format and options.

      • The identifier is immutable after creation.

      Example

      const identifier = IdentifierUtilities.generateIdentifier({
         format: "Name",
         options: {
           prefix: "prefix-",
           suffix: "-suffix",
           length: 10
         }
