    Enumeration BaseClientErrorCodes

    BaseClientErrorCodes is an enum that defines the common HTTP status codes for client error responses. These codes are used to indicate that the client seems to have made an error. The codes follow the web standard for client error codes. See: https://www.iana.org/assignments/http-status-codes/http-status-codes.xhtml
